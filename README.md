# 📈 Market Insight Daily - 股市智能分析系统

> **基于 AI 驱动的 A股/港股/ETF 每日自动分析与决策支持系统**

## 🌟 系统简介

**Market Insight Daily** 是一套专为投资者设计的自动化股市分析工具。系统每日收盘后（18:00）自动运行，抓取真实市场行情，结合核心技术指标（MA, MACD, RSI）与 AI 决策逻辑，生成结构化的分析报告，并同步推送到您的飞书群及本 GitHub Pages 永久站点。

## 🚀 核心功能

- **多市场覆盖**：支持 A股、港股及主流 ETF 的实时数据追踪。
- **技术面深挖**：自动计算 5/20 日均线、RSI 相对强弱指标及 MACD 趋势指标。
- **AI 决策仪表盘**：内置交易纪律，针对每只标的提供“买入/观望/卖出”建议及决策依据。
- **双端访问**：飞书结构化消息推送 + 响应式 HTML 报告，支持手机与电脑端流畅查看。
- **配置灵活**：支持通过 `config.json` 自由增减关注标的，无需修改核心代码。

## 📊 关注标的示例

| 类型 | 数量 | 代表标的 |
| :--- | :--- | :--- |
| **A股** | 10只 | 宁德时代、紫金矿业、亿纬锂能等 |
| **港股** | 2只 | 宁德时代H、紫金矿业H |
| **ETF** | 6只 | 软件ETF、黄金ETF、红利低波ETF等 |

## 🛠️ 如何自定义配置

您可以直接修改项目中的 `config.json` 文件来管理您的自选股：

```json
{
    "stocks": {
        "A_STOCKS": ["601112.SS", "300014.SZ", ...],
        "HK_STOCKS": ["3750.HK", ...],
        "ETFS": ["518880.SS", ...]
    },
    "indices": ["000001.SS", "399001.SZ", "399006.SZ"]
}
```

## 📅 自动化流程

1. **15:00 - 17:30**：等待收盘数据清算。
2. **18:00**：触发 Manus AI 分析任务。
3. **18:05**：生成 HTML 报告并推送到 GitHub Pages。
4. **18:10**：发送飞书卡片消息至指定群组。
